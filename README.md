# 🐶 犬と暮らすInstagram投稿アシスタント

## セットアップ手順

### 1. 依存パッケージのインストール
```bash
npm install
```

### 2. ローカルで動かす
```bash
npm run dev
```

### 3. ビルド（Vercelにデプロイする前に確認）
```bash
npm run build
```

## Vercelへのデプロイ
1. このフォルダをGitHubリポジトリにアップロード
2. [vercel.com](https://vercel.com) でGitHubと連携
3. リポジトリを選択して「Deploy」を押すだけ！
